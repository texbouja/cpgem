Activité 2.19 - Espérance d’une forme quadratique.
Soit \(X \) un vecteur aléatoire de \(\mathbb{R}^n \) ayant un moment d’ordre \(2\text{.}\) On note \(\mathbf{\Sigma}(X) \) la matrice des covariances de \(X \) et \(\mu\) son espérance.
L’activité propose de montrer la formule (2.2) dite formule des formes quadratiques et quelques exemples d’application ensuite.
(a) Formule des formes quadratiques.
Soit \(A\) une matrice carrée constante. Démontrer la formule :
\begin{equation} \EE({}^tX A X) = \text{tr}(A \mathbf{\Sigma}(X)) + {}^t\mu A \mu.\tag{2.2} \end{equation}
Solution.
Soit \(X\) un vecteur aléatoire de dimension \(n\text{,}\) de moyenne \(\mu = \EE(X)\) et de matrice de covariance \(\mathbf{\Sigma}(X) = \EE((X - \mu)(X - \mu)^\top)\text{.}\) Soit \(A\) une matrice carrée de taille \(n \times n\text{.}\)
Nous voulons calculer \(\EE({}^tX A X)\text{.}\) Pour cela, décomposons \(X\) en sa moyenne \(\mu\) et son écart par rapport à la moyenne \(Y = X - \mu\text{.}\) Ainsi, \(X = \mu + Y\text{,}\) où \(\EE(Y) = 0\) et \(\mathbf{\Sigma}(X) = \EE(Y Y^\top)\text{.}\)
En substituant \(X = \mu + Y\) dans la forme quadratique, nous obtenons :
\begin{equation*} {}^tX A X = (\mu + Y)^\top A (\mu + Y). \end{equation*}
Développons cette expression :
\begin{equation*} {}^tX A X = \mu^\top A \mu + \mu^\top A Y + Y^\top A \mu + Y^\top A Y. \end{equation*}
Prenons l’espérance des deux côtés. Comme \(\EE(Y) = 0\text{,}\) les termes linéaires en \(Y\) disparaissent :
\begin{equation*} \EE({}^tX A X) = \mu^\top A \mu + \EE(Y^\top A Y). \end{equation*}
Il reste à calculer \(\EE(Y^\top A Y)\text{.}\) Notons que \(Y^\top A Y\) est un scalaire, donc égal à sa trace :
\begin{equation*} Y^\top A Y = \text{tr}(Y^\top A Y). \end{equation*}
En utilisant la propriété de cyclicité de la trace (\(\text{tr}(ABC) = \text{tr}(CAB)\)), nous avons :
\begin{equation*} Y^\top A Y = \text{tr}(A Y Y^\top). \end{equation*}
En prenant l’espérance, nous obtenons :
\begin{equation*} \EE(Y^\top A Y) = \EE(\text{tr}(A Y Y^\top)) = \text{tr}(A \EE(Y Y^\top)). \end{equation*}
Or, \(\EE(Y Y^\top) = \mathbf{\Sigma}(X)\text{,}\) donc :
\begin{equation*} \EE(Y^\top A Y) = \text{tr}(A \mathbf{\Sigma}(X)). \end{equation*}
En combinant ce résultat avec l’expression précédente, nous obtenons finalement :
\begin{equation*} \EE({}^tX A X) = \mu^\top A \mu + \text{tr}(A \mathbf{\Sigma}(X)). \end{equation*}
Ce qui conclut la démonstration.
(b) Modèle de portefeuille binaire.
Objectif : Comprendre comment les formes quadratiques peuvent modéliser les interactions entre actifs dans un portefeuille binaire.
Vous gérez un portefeuille de 3 actifs binaires. Chaque actif \(X_i\) peut être actif (\(X_i = 1\)) avec une probabilité \(p_i\) ou inactif (\(X_i = 0\)) avec une probabilité \(1-p_i\text{.}\) Les actifs peuvent être corrélés.
Données :
  • \(X_1 \sim \mathscr B(0.5)\text{,}\)
  • \(X_2 \sim \mathscr B(0.6)\text{,}\)
  • \(X_3 \sim \mathscr B(0.7)\text{.}\)
  • La corrélation entre \(X_1\) et \(X_2\) est \(\rho = 0.3\text{,}\) et \(X_3\) est indépendant des deux autres.
  • La matrice de pondération \(A = \begin{pmatrix} 1 \amp 0.5 \amp 0 \\ 0.5 \amp 1 \amp 0 \\ 0 \amp 0 \amp 1 \end{pmatrix}\text{.}\)
Rôle de la matrice de pondération.
La matrice \(A\) représente les interactions entre les actifs du portefeuille. Chaque élément \(A_{ij}\) quantifie l’influence de l’actif \(j\) sur l’actif \(i\text{.}\) Par exemple, \(A_{12} = 0.5\) signifie que l’actif 2 influence l’actif 1 avec un poids de 0.5. Les éléments diagonaux \(A_{ii}\) représentent l’importance intrinsèque de chaque actif.
Tâches :
  1. Calculez la matrice de covariance \(\Sigma\text{.}\)
  2. Calculez \(\mathbb{E}(X^\top A X)\) en utilisant la formule \(\text{Tr}(A \Sigma) + \mu^\top A \mu\text{.}\)
  3. Interprétez le résultat en termes de risque du portefeuille.
Solution.
Solution :
  1. Matrice de covariance : \(\Sigma = \begin{pmatrix} 0.25 \amp 0.09 \amp 0 \\ 0.09 \amp 0.24 \amp 0 \\ 0 \amp 0 \amp 0.21 \end{pmatrix}. \)
  2. Calcul de \(\mathbb{E}(X^\top A X)\) :
    • \(\text{Tr}(A \Sigma) = 0.79\text{,}\)
    • \(\mu^\top A \mu = 1.47\text{,}\)
    • \(\mathbb{E}(X^\top A X) = 0.79 + 1.47 = 2.26\text{.}\)
  3. Interprétation : Une espérance plus élevée indique un portefeuille plus risqué.
(c) Réseau aléatoire avec interactions binaires.
Objectif : Explorer comment les formes quadratiques peuvent modéliser les connexions dans un réseau aléatoire.
Vous étudiez un réseau de 4 nœuds, où chaque nœud \(X_i\) est actif (\(X_i = 1\)) avec une probabilité \(p_i\) ou inactif (\(X_i = 0\)) avec une probabilité \(1-p_i\text{.}\) La matrice \(A\) représente les connexions entre les nœuds.
Données :
  • \(X_1 \sim \mathscr B(0.5)\text{,}\)
  • \(X_2 \sim \mathscr B(0.6)\text{,}\)
  • \(X_3 \sim \mathscr B(0.7)\text{,}\)
  • \(X_4 \sim \mathscr B(0.8)\text{.}\)
  • les variables \(X_1,X_2,X_3,X_4\) sont supposés inépendantes.
  • La matrice de connexion \(A = \begin{pmatrix} 0 \amp 1 \amp 0 \amp 1 \\ 1 \amp 0 \amp 1 \amp 0 \\ 0 \amp 1 \amp 0 \amp 1 \\ 1 \amp 0 \amp 1 \amp 0 \end{pmatrix}\text{.}\)
Rôle de la matrice de connexion.
La matrice \(A\) représente les connexions entre les nœuds du réseau. Chaque élément \(A_{ij}\) vaut 1 si les nœuds \(i\) et \(j\) sont connectés, et 0 sinon. Par exemple, \(A_{12} = 1\) signifie que les nœuds 1 et 2 sont connectés. Les éléments diagonaux \(A_{ii}\) sont fixés à 0 pour indiquer qu’un nœud n’est pas connecté à lui-même.
Tâches :
  1. Calculez \(\mathbb{E}(X^\top A X)\) en utilisant la formule \(\text{Tr}(A \Sigma) + \mu^\top A \mu\text{.}\)
  2. Interprétez le résultat en termes de nombre attendu de paires de nœuds actifs connectés.
Solution.
Solution :
  1. Calcul de \(\mathbb{E}(X^\top A X)\) :
    • La matrice de covariance \(\mathbf{\Sigma}(X)\) est diagonale car les variables sont indépendantes :
      \begin{equation*} \mathbf{\Sigma}(X) = \begin{pmatrix} 0.25 \amp 0 \amp 0 \amp 0 \\ 0 \amp 0.24 \amp 0 \amp 0 \\ 0 \amp 0 \amp 0.21 \amp 0 \\ 0 \amp 0 \amp 0 \amp 0.16 \end{pmatrix}. \end{equation*}
    • \(\text{Tr}(A \Sigma) = 0\) (car \(A\Sigma\) a des zéros sur la diagonale),
    • \(\mu^\top A \mu = 3.42\text{,}\)
    • \(\mathbb{E}(X^\top A X) = 0 + 3.42 = 3.42\text{.}\)
  2. Interprétation : En moyenne, 3.42 paires de nœuds actifs sont connectées dans le réseau.
(d) Modèle de vote avec dépendances.
Objectif : Analyser comment les formes quadratiques peuvent modéliser les influences entre électeurs dans un système de vote.
Vous étudiez un système de vote avec 3 électeurs. Chaque électeur \(X_i\) vote pour un candidat (\(X_i = 1\)) avec une probabilité \(p_i\) ou s’abstient (\(X_i = 0\)) avec une probabilité \(1-p_i\text{.}\) Les votes peuvent être corrélés.
Données :
  • \(X_1 \sim \mathscr B(0.6)\text{,}\)
  • \(X_2 \sim \mathscr B(0.7)\text{,}\)
  • \(X_3 \sim \mathscr B(0.8)\text{.}\)
  • La corrélation entre \(X_1\) et \(X_2\) est \(\rho = 0.4\text{,}\) et \(X_3\) est indépendant des deux autres.
  • La matrice de pondération \(A = \begin{pmatrix} 1 \amp 0.5 \amp 0 \\ 0.5 \amp 1 \amp 0 \\ 0 \amp 0 \amp 1 \end{pmatrix}\text{.}\)
Rôle de la matrice de pondération.
La matrice \(A\) représente les influences entre les électeurs. Chaque élément \(A_{ij}\) quantifie l’influence de l’électeur \(j\) sur l’électeur \(i\text{.}\) Par exemple, \(A_{12} = 0.5\) signifie que l’électeur 2 influence l’électeur 1 avec un poids de 0.5. Les éléments diagonaux \(A_{ii}\) représentent l’importance intrinsèque de chaque électeur.
Tâches :
  1. Calculez la matrice de covariance \(\Sigma\text{.}\)
  2. Calculez \(\mathbb{E}(X^\top A X)\) en utilisant la formule \(\text{Tr}(A \Sigma) + \mu^\top A \mu\text{.}\)
  3. Interprétez le résultat en termes d’influence des électeurs sur le résultat du vote.
Solution.
Solution :
  1. Matrice de covariance : \(\Sigma = \begin{pmatrix} 0.24 \amp 0.112 \amp 0 \\ 0.112 \amp 0.21 \amp 0 \\ 0 \amp 0 \amp 0.16 \end{pmatrix}. \)
  2. Calcul de \(\mathbb{E}(X^\top A X)\) :
    • \(\text{Tr}(A \Sigma) = 0.722\text{,}\)
    • \(\mu^\top A \mu = 1.97\text{,}\)
    • \(\mathbb{E}(X^\top A X) = 0.722 + 1.97 = 2.692\text{.}\)
  3. Interprétation : L’espérance élevée reflète une forte influence des électeurs corrélés sur le résultat du vote.
